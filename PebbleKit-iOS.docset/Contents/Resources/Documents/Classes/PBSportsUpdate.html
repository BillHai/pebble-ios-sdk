PBSportsUpdate Class Reference

Inherits from NSObject
Declared in PBWatch+Sports.h

Overview

Instances of this convenience class represent an update of the current time, distance and pace, and has a method to convert the values into a dictionary that can be used with -sportsAppUpdate:onSent:.
